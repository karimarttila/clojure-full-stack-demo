Simple Server

This is the Simple Server. Available commands are:

  • GET / - The Frontend.
  • GET /info.html - This file.
  • GET /api/info - Gives general information.
  • GET /api/ping - Ping test.
  • POST /api/ping - Ping test (expects to find parameter ping in body).
  • POST /api/sign-in - Creates a new user.
  • POST /api/login - Logs on to service using credentials created using sign-in. You get json web token that must be supplied in following requests.
  • GET /api/products-groups - Gets product groups.
  • GET /api/products/ - Gets products in product group.
  • GET /api/product/ - Gets specific product information.
NOTE: You can see examples how to call these endpoints in the scripts directory (using curl).
